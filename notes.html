我想要圖片跟文字各佔一半！-Grid Layout
當我想要讓圖片和文字在同個div裡，我希望它各佔一半時：

首先，圖片先設定寬度大小

其次，將它們所屬的那個div（父元素）設定為

display:grid;

grid-template-columns: 1fr 1fr;

其中，fr表示單位，分數的概念，有兩個1fr則表示，我要把這個div切成1/2,1/2。如果有三個1fr則表示每一塊佔1/3，以此類推。

1.到https://github.com/michalsnik/aos ，複製安裝AOS的的程式碼到“標籤head裡“和“標籤body 結尾之前“。
2.在https://michalsnik.github.io/aos/裡，選擇你要的樣式，然後把data-aos那段複製貼上。表示單位，分數的概念，有兩個1fr則表示，我要把這個div切成1/2,1/2。如果有三個1fr則表示每一塊佔1/3，以此類推。

margin.padding.border
margin:使元素跟元素之間不要靠太近（外間距）
padding:使元素和內容不要靠太近（內間距）
border:外圍 box-sizing:調整內距和標框的計算方式，又包含content-box（僅包含內容寬度）和border-box（包含padding和border））

寫法：

上、右、下、左 ex. margin: 20px 15px 10px 30px;
上、左右、下 ex. padding: 100px auto 20px;
上下、左右 ex. margin: 20px 15px ;
上下左右 padding: 20px;
display:none v.s. visibility:hidden
display:none 隱藏且不佔空間
visibility:hidden 隱藏且佔空間
一樣高的col(不受內容文字多寡影響)
父層：display: flex;
子元素：flex: 1; （使子元素擴展以填滿可用空間）
用Grid Layout設計出3欄（行）2列，並且具RWD效果
首先，欄位有三行，列有兩列，其中某一個item是佔了兩列。
所以利用Grid Layout時，display：grid; grid-template-columns利用百分比去寫你要的三欄佔比大小，並且利用grid-gap去調整間距。

其次，怎麼將其中一個item變成涵蓋兩列？

先找到「它是第幾欄」，比如是第三欄那就寫grid-column:3;然後要把這個item擴展把列變寬-- grid-row:1/span;
其中，因為只有那個item要調整，所以寫1;利用span進行擴展，擴展為兩列，則寫2;

再來，怎麼變成RWD？利用@media去調整。裡面寫：

grid-template-columns:auto;
grid-column:1;
grid-row:5;

var v.s. let v.s. const

可否重複宣告？

  • let和const不可以重複宣告
  • var可以重複宣告

使用前是否要先宣告？（也就是宣告時就要給值了嗎？）

  • let和const使用前就要先宣告

是否會被綁定到this？

  • let和const不會被綁定到this

是否會被提升？

  • let和const不會被提升，因此在作用域中，變數通常用let或是const進行宣告
null v.s. undefined
  • 如果變數沒有提供初始值，而你要使用這個變數：undefined
  • 如果要使用一個物件的屬性或是元素，而該屬性或元素不存在：undefined
  • 該傳參數的回調函數沒有傳參數：undefined
  • 使用void：undefined
  • null的用意是用來宣告「空值」
  • DOM操作時，若DOM元素不存在：null

null==undefined，return true

null===undefined，return false
